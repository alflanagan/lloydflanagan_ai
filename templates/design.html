Design Notes

Design Goals

This site is built around a single guiding idea: prove that a complete, production-ready personal website can be created entirely through natural-language prompts to an AI, with no hand-written code. Beyond that experiment, the goals are simplicity and readability — a clean layout that stays out of the way of the content, fast page loads using CDN-hosted assets, a consistent visual language throughout via a shared component library, and a responsive design that works comfortably on any screen size.

Third-Party Software

  • FastAPI — Modern Python web framework chosen for its async-native design, automatic validation via Pydantic, and clean decorator-based routing.
  • Uvicorn — Lightweight ASGI server that pairs naturally with FastAPI and supports hot-reload during development.
  • Jinja2 — Templating engine integrated directly into FastAPI; allows HTML pages to be composed from a shared base layout with named blocks.
  • Shoelace — Web-components UI library that provides polished, accessible elements (cards, badges, buttons) without locking the project into a JavaScript framework.
  • Lit — Minimal library used to build the custom <site-header> web component; chosen because it compiles down to standard custom elements with no runtime overhead beyond the small Lit core.
  • Google Fonts — Supplies the Playwrite New Zealand Basic typeface used in the site heading, loaded via CDN for zero self-hosting complexity.
  • uv — Fast Python package and project manager used to install dependencies and run scripts, replacing the traditional pip/virtualenv workflow.
  • Fly.io — Deployment platform that hosts the containerised app globally; chosen for its simple CLI, free tier, and built-in HTTPS.
